## Description

Enian Combat Backend is built with [Nest](https://github.com/nestjs/nest), a progressive Node.js framework, to power our GameFi project on the TON blockchain. This service manages game logic, user data, and blockchain interactions.

## Installation

```bash
# install dependencies
$ npm install

# setup env
$ cp .env.example .env

# prisma run migration
$ npx prisma migrate deploy

# run db seed
$ npm run db:seed

# run single db seed
$ npm run db:seed --file=write_file_name_here
```

## Set Up Other Dependencies

```bash
$ npm run postinstall
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
